# AI Agent Entry Point

This repository uses a centralized AI instruction file.

All AI coding agents, assistants, and automation tools should follow:

- .github/copilot-instructions.md

If there is any conflict between instructions, the contents of:

.github/copilot-instructions.md

take precedence.

## Primary Responsibilities

When making changes:

- Follow existing architecture.
- Follow TypeScript best practices.
- Follow security requirements.
- Respect role-based authorization.
- Prefer simple and maintainable solutions.
- Keep the application mobile-first.
- Avoid introducing unnecessary dependencies.

## Important

Before implementing any feature:

1. Check if a similar implementation already exists.
2. Reuse existing components whenever possible.
3. Follow naming conventions.
4. Verify responsive behavior.
5. Verify role permissions.
6. Verify error handling.
7. Verify TypeScript type safety.

For complete project guidance see:

.github/copilot-instructions.md
